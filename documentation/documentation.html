C) JavaScript - Go to top

These are the JS files that are loaded into templates in end of the Body Section. Only Modernizr.js is loaded in Head Section.

This theme imports three types of Javascript files:

  1. jQuery - a Javascript library that greatly reduces the amount of code that must be written.

  2. My custom scripts. You can find the custom script in the script.js file.

  3. Plugins. In addition to the custom scripts, I implement a few plugins to create the effects. You can find the script of some plugins in the plugins.js.

    4. These are the rest of the scripts used in the template with their related link for documentation.

    • Modernizr - a JavaScript library that detects HTML5 and CSS3 features in the userâ€™s browser.
    • Respond - a JavaScript plugin to responsive web designs in browsers that don't support CSS3 Media Queries - Internet Explorer 8 and under.
    • HTML5Shiv - a JavaScript plugin to to enable use of HTML5 sectioning elements in Internet Explorer.
    • Twitter Bootstrap 3 - one of the most popular front-end development frameworks in the world.
    • SuperSized - a fullscreen background slideshow built using the jQuery library.
    • jPreloader - a Javascript library that lets you manage and co-ordinate the loading of assets.
    • jQuery Easing - a jQuery plugin for advanced easing options.
    • Waypoints - a jQuery plugin that makes it easy to execute a function whenever you scroll to an element.
    • Stellar - a jQuery plugin that provides parallax scrolling effects to any scrolling element.
    • Fancybox - a plugin for displaying images, html content and multi-media in a Mac-style "lightbox" that floats overtop of web page.
    • Wow - a jQuery plugin for revealing CSS animation as you scroll down a page.
    • Isotope Filter - a jQuery plugin for hide and revealing item elements.

D) Slider settings - Go to top

This is the HTML syntax for the slider.

You can find the CSS for the slider in the third section of style.css file.

You can find the JS script that affects the slider in the beginning of the script.js file. You can change the default parameters of the slider. Enter the path for images, insert text, a thumb and a URL for a possible link.

For more information I recomand you to read the documentation on Supersized Plugin.

E) Text and images customization - Go to top

In HTML file, you can easily change the text between the p or the h tags.

You can change the images in the HTML file as well by typing the source of you image files. Exemple: img src="img/portfolio/thumbs/1.jpg"

For social media customization replace the # symbol with the social media link. Exemple: www.twitter.com/username.
